angular.json - Contains Angular CLI configuration for project building, serving, testing, and production budgets.

tsconfig.json - Contains the base TypeScript compiler configuration for the project.

tsconfig.app.json - Contains TypeScript compiler settings specifically for the Angular application.

package.json - Contains project dependencies, development dependencies, and npm scripts.

src/main.ts - The entry point of the Angular application that bootstraps the Angular application.

src/app/app.config.ts - Contains application-level configuration and providers for the standalone Angular application.

src/app/app.ts - Defines the root component of the Angular application.

src/index.html - The main HTML document containing the app-root element where Angular renders the application.
